## How to Play
1. Launch the game.
2. Enter coordinates in the format `A1` (ColumnLetter + RowNumber).
3. Hit (`X`) or miss (`O`) enemy ships.
4. The AI takes its turn.
5. Continue until all enemy ships are destroyed (or yours are!).
